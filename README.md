# CV Global Mandiri Mobilindo
### Telkomsel · IndiHome · Telkom Enterprise Partner

Website profil korporasi resmi **CV Global Mandiri Mobilindo (GMM)** — mitra enterprise Telkomsel, IndiHome, dan Telkom Indonesia untuk wilayah **Sulawesi Utara & Maluku Utara**.

🌐 **Live Site:** [gmm-telkom.vercel.app](https://gmm-telkom.vercel.app)

---

## Fitur

| Seksi | Deskripsi |
|---|---|
| **Beranda** | Hero section dengan statistik kunci perusahaan |
| **Profil Korporasi** | Visi, misi, dan latar belakang GMM |
| **Wilayah WOK** | Peta & info coverage area per wilayah operasi |
| **Struktur Organisasi** | Direktori tim manajemen & team leader |
| **Kinerja & KPIs** | Dashboard target dan pencapaian kinerja |
| **Roadmap Strategi** | Rencana strategis dan target ke depan |

## Teknologi

- **HTML5** — Single-page application
- **CSS3** — Glassmorphism, animasi, responsive design
- **JavaScript** — Navigasi, scroll behavior, interaksi UI
- **Motion (v12)** — Animasi halaman
- **Google Fonts** — Plus Jakarta Sans, Poppins, Inter
- **Vercel** — Hosting & deployment

## Menjalankan Secara Lokal

```bash
# Clone repo
git clone https://github.com/greendayiezx/GMM-Telkom.git
cd GMM-Telkom

# Buka langsung di browser (tidak perlu build)
start index.html
```

Atau gunakan live server (misal VS Code Live Server extension) untuk pengalaman terbaik.

## Deployment

Project ini ter-deploy otomatis ke Vercel setiap kali ada push ke branch `main`.

```bash
# Deploy manual (butuh Vercel CLI)
vercel --prod
```
